🌕️lua-re: OCaml bindings for Lua
===

OCaml bindings for Lua through ctypes.
Currently supports Lua 5.2 only.

# Installation
<!-- $MDX skip -->
```bash
$ git clone https://github.com/Nymphium/ocaml-lua-re
$ direnv allow
$ dune build
```
